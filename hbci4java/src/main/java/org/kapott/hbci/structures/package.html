In diesem Package befinden sich einige Klassen, die Datenstrukturen zum besseren Handling oft wiederkehrender Elemente repräsentieren.

Diese Klassen sind prinzipiell nicht nur auf den Einsatz in Deutschland (Länderkennzeichen DE) bzw. in Ländern mit der Währung EURO (Währungskennzeichen EUR) beschränkt. Die genannten Werte stellen jedoch die default-Werte dar, wenn keine anderen Daten angegeben werden.
